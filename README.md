# Git Flow - Testes

## - git clone: clona um respositório remoto para o repositório local

## - git init: inicializa as configurações de versionamento do git no prjeto

## - git add: adiciona arquivos ao repositório local antes de efetuar o envio

## - git status: exibe os arquivos adicionados ou não ao repositório local

## - git commit: reune as modificações dos arquivos adicionados e adiciona uma mensagem

## - git push: envia as alterações do repositório local para o remoto

## - git pull: trás as alterações do repositório remoto para o local

## - git checkout -b NOME_DA_BRANCH: cria uma nova branch e vai para ela

## - git checkout NOME_DA_BRANCH / git switch NOME_DA_BRANCH: muda de branch

## - git branch: lista as branchs locais

## - git branch -a: lista todas as branchs

## - git remote add NOME_DA_CONEXÃO LINK_DO_REPOSITORIO: conecta o projeto com o repositório remoto

## - git merge NOME_DA_BRANCH_QUE_VIRÁ_AS_MODIFICAÇÕES: mescla uma branch na outra

## - git stash: salva as alterações em uma pilha a parte

## - git stash pop: aplica as modificações e remove da pilha

## - git stash apply: aplica as modificações, mas permanece na pilha
